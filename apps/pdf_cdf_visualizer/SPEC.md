# 仕様書：PDF・CDF・区間確率ビジュアライザ

## 1. 教育上の狙い（なぜ作るか）

学習者の最大のつまずきは、**連続確率変数で「密度の高さ」と「確率」を混同する**ことです。
- 「$f(x)$ が大きい＝その値が出る確率が高い」までは良いが、「$f(x)$ が確率の値そのもの」と誤る。
- $P(X=x)=0$ を「ほぼ0だが正」と誤る。
- PDF と CDF を別々の暗記対象として切り離す。

このツールは、**同じ確率を「PDF の面積」と「CDF の差」として並べて見せ**、さらに
**区間を縮める実験**で「点の確率は0／密度は残る」を体験させることで、これらを一度に解消します。

## 2. 対応する誤解と、ツールでの解消法

| 誤解 | ツールでどう壊すか |
|---|---|
| 密度＝確率 | 高さ $f(x_0)$ を橙線で表示し、「これは確率ではない」と明示。面積（緑）と並置。 |
| 密度は1以下 | σ を小さく／一様を狭めると高さが1を超える様子を観察。 |
| $P(X=x)>0$ | ε→0 で区間確率が0へ向かう数値を3列メトリクスで提示。 |
| PDF と CDF は別物 | 左にPDF面積、右にCDF差を同時表示。同じ数値が2通りで出る。 |
| 面積とCDF差が別 | 数値積分（面積）と $F(b)-F(a)$ を両方計算し、差 $\approx 0$ を毎回表示。 |

## 3. 機能要件

- 分布選択：正規・対数正規・指数・一様（`core.DISTRIBUTIONS` で定義、追加容易）。
- パラメータはスライダーで連続変更。台（描画範囲）は分位点 `ppf(0.001..0.999)` で自動決定。
- 問い方：区間 $P(a\le X\le b)$／左裾 $P(X\le b)$／右裾・超過 $P(X>a)$。
- 確率を **3通り**（CDF差・数値積分の面積・両者の差）で表示し一致を可視化。
- 点プローブ：$x_0$ と $\varepsilon$ で「高さ $f(x_0)$」「区間確率」「近似 $f(x_0)\cdot2\varepsilon$」を比較。
- 解説（学習目標／観察点／対応式／誤解／ミニ演習＋ヒント／電力接続）をエクスパンダで併設。

## 4. 非機能要件

- **数理はUIから分離**（`core.py`）。`python core.py` で自己テスト可能。
- Streamlit 非依存の単体検証ができること（CI 化の素地）。
- 1画面で完結（左右2図＋メトリクス）。スマホ閲覧は対象外。

## 6. 数学的対応（実装の根拠）

$$F(x)=\int_{-\infty}^x f,\quad f=\frac{dF}{dx},\quad P(a\le X\le b)=\int_a^b f=F(b)-F(a),\quad P(X=x)=0.$$

実装：
- `core.interval_prob_cdf` … $F(b)-F(a)$（`scipy` の `cdf`）
- `core.interval_prob_area` … 台形則による $\int_a^b f$
- `core.point_interval_demo` … 高さ・区間確率・$f(x_0)\cdot2\varepsilon$

## 7. 既知の限界・仮定

- 連続分布のみ（離散 PMF は別ツール／Module 2 本文で扱う）。
- 多変量・条件付き分布は対象外（→ Module 3、散布図ツールへ）。
- 台の自動決定は分位点ベースのため、極端な裾をもつ設定では端が切れる。

## 8. 拡張案（次の反復）

1. **離散モード**：二項・ポアソンの PMF を棒で表示し「質量＝高さ」を対比（連続との違いを強調）。
2. **分位点プローブ**：CDF を逆引きして $q_\alpha$（VaR）を表示、Module 3/6 への橋渡し。
3. **2分布比較**：正規 vs 対数正規を重ね、同じ平均でも右裾の超過確率が桁違いになる様子。
4. **データ重ね描き**：CSV のヒストグラム/ECDF を重ね、Module 4（データ→分布）へ接続。
5. **CSV/PNG エクスポート**：観察結果を演習解答に貼れるように。
