# 仕様書：確率的最適化コンパレータ

## 1. 教育上の狙い

本教材の「完成の定義」を満たすツール：
> 学習者が**同じ問題**を 決定論／期待値／ロバスト／チャンス制約／CVaR／分布ロバスト の各形式で比較し、
> **「なぜこの形式を選ぶか」を自分の言葉で説明できる**状態にする。

そのために、**決定 $x^\*$ が形式で大きく変わる**様子（本例 100〜137）を1画面で見せ、
各形式が「自分の目的では最良・他指標では譲る」ことを評価表で可視化する。

## 2. 機能要件
- 統一問題（newsvendor 型）を6形式で解く（`core.solve_all`）。
- パラメータ：μ, σ, 分布種別, c_s, c_o, ε, α, ロバスト幅 k, シナリオ数 N。
- 出力：①需要分布上の決定マップ（x* 縦線）②x* 棒グラフ（保守性順）③評価表（x*, E[C], 不足確率, CVaR, ねらい）。
- 各指標のベスト形式をハイライト（②=E[C]最小, ⑤=CVaR最小, ④≈ε）。
- 解説：学習目標／選び方／数式／誤解／電力接続。

## 3. 非機能要件
- 数理を UI から分離（`core.py`、`python core.py` で自己テスト）。
- cvxpy 非搭載環境でも CVaR をグリッド探索でフォールバック（落ちない）。
- 1画面で完結。

- [x] `core.py` self-test：x* の順序 ①<②<⑤<③、②≈120, ③≈136.8, ①=100、**②が E[C] 最小・⑤が CVaR 最小**。

## 5. 数理的根拠（実装）
| 形式 | 実装 |
|---|---|
| ①決定論 | $x=\mu$ |
| ②期待値 | `dist.ppf(cs/(cs+co))`（臨界分位点） |
| ③ロバスト | $(c_s(\mu+k\sigma)+c_o(\mu-k\sigma))/(c_s+c_o)$ |
| ④チャンス | `dist.ppf(1-eps)` |
| ⑤CVaR | cvxpy: $\min\ \eta+\frac{1}{(1-\alpha)S}\sum u_i,\ u_i\ge C_i-\eta$ |
| ⑥DRO | Scarf: $\mu+\frac{\sigma}{2}(\sqrt{c_s/c_o}-\sqrt{c_o/c_s})$ |

評価は共通サンプルで E[C]・不足確率・CVaR を算出（公平な採点）。

## 6. 既知の限界・仮定
- 単段・1変数（多段・多変数・ネットワーク制約は対象外）。
- ロバスト集合は対称区間 $[\mu\pm k\sigma]$（一般の多面体・楕円体は拡張案）。
- DRO は Scarf（平均・分散）のみ。Wasserstein 球は拡張案。
- 連続分布のみ（離散シナリオ直接入力は拡張案）。

## 7. 拡張案（次の反復）
1. **二段階リコース**版（第1段＋第2段 y(ξ)）を追加し、リコース価値を可視化。
2. ロバストを Bertsimas–Sim 予算 Γ で連続調整。
3. DRO を Wasserstein 半径 ε で連続調整し、データ量との関係を見せる。
4. 蓄電池運用（多時間帯・容量/SoC 制約）への拡張（`apps/` 新ツール）。
5. 結果の CSV/PNG エクスポート（演習解答へ貼付）。
