# 用語集

主要用語を**1行定義＋出てくるモジュール**で引けるように。詳しくは各 Module へ。

## 確率の言語（Module 1–2）

| 用語 | 1行定義 | 章 |
|---|---|---|
| 標本空間 $\Omega$ | 起こりうる結果すべての集合。 | M1 |
| 事象 | 結果の部分集合（「起こる/起こらない」）。 | M1 |
| 確率変数 $X$ | 結果に数を割り当てる関数。 | M2 |
| PMF $p(x)$ | 離散変数の「各値の確率」。棒の高さ＝確率。 | M2 |
| PDF $f(x)$ | 連続変数の**密度**。高さは確率ではない（1を超えてよい）。 | M2 |
| CDF $F(x)$ | $P(X\le x)$。**確率＝区間の面積＝CDFの差**。 | M2 |
| 分位点 $F^{-1}(p)$ | 「下から $p$ 割」の値。チャンス制約・CVaRの主役。 | M2,M6 |

## 条件付き・ベイズ（Module 1）

| 用語 | 1行定義 | 章 |
|---|---|---|
| 条件付き確率 $P(A\mid B)$ | $B$ が分かった上での $A$ の確率。 | M1 |
| ベイズの定理 | 事前 × 尤度 → 事後。情報で確率を更新。 | M1 |
| 基準率の誤謬 | 稀な事象では「陽性≠本物」。事前を無視する誤り。 | M1 |

## 要約とリスク（Module 3）

| 用語 | 1行定義 | 章 |
|---|---|---|
| 期待値 $E[X]$ | 確率で重みづけた平均。 | M3 |
| 分散 / 標準偏差 | ばらつきの大きさ。$\sigma=\sqrt{\mathrm{Var}}$。 | M3 |
| 共分散 / 相関 | 2量の連動。相関は合算の分散を増減させる。 | M3 |
| 条件付き期待値 | 情報を得た後の平均。最良予測。 | M3 |

## 不確実性の種類（Module 0,4）

| 用語 | 1行定義 | 章 |
|---|---|---|
| 偶然的不確実性 (aleatory) | 本質的な揺れ（ばらつき）。情報では**減らない**。 | M0 |
| 認識論的不確実性 (epistemic) | 知らないこと（誤差）。情報で**減る**。 | M0,M4 |
| 標本 / 推定 | データは真の分布から漏れた有限の標本。推定値もばらつく。 | M4 |

## 計算（Module 5）

| 用語 | 1行定義 | 章 |
|---|---|---|
| モンテカルロ法 | 乱数で期待値・積分を近似。誤差は $1/\sqrt N$。 | M5 |
| 大数の法則 / 中心極限定理 | 平均は真値へ収束 / その揺れは正規に近づく。 | M5 |
| シナリオ | 分布の有限な代表点＋確率。離散近似であって手抜きではない。 | M5 |
| SAA（標本平均近似） | 期待値最適化を有限シナリオの平均で置換して解く。 | M5,M6 |

## 不確実性下の最適化（Module 6, 6b）

| 用語 | 1行定義 | 章 |
|---|---|---|
| 決定論的最適化 | 不確実量を1点（平均等）で固定して解く。 | M6 |
| 期待値最適化 | 期待コストを最小化。平均的に良い。 | M6 |
| ロバスト最適化 | 不確実性**集合**の中の最悪に備える。 | M6 |
| チャンス制約 | 制約違反確率を $\varepsilon$ 以下に抑える＝分位点。 | M6 |
| VaR / CVaR | VaR＝分位点の損失。CVaR＝その先（裾）の条件付き平均。 | M6 |
| 分布ロバスト | 分布を1つに決めず、分布の**族**の最悪に備える（例：Scarf）。 | M6 |
| 不確実性集合 | 「ありうる実現の範囲」。ロバスト最適化の入力。 | M6 |
| newsvendor / 臨界比 | 「何個確保」の原型。最適は $c_s/(c_s+c_o)$ 分位点。 | M6 |
| recourse（リコース） | 不確実性が判明した後に取れる修正行動。 | M6b |
| 二段階確率計画 | いま決め→観測→直す、の二段階。 | M6b |
| VSS | 確率的に解く価値＝EEV−確率計画の差。 | M6b |
| EVPI | 完全情報の価値＝確率計画−WS の差。 | M6b |
