# 演習（中級）★★ — Module 1

> 学習目標：ベイズで基準率の効果を計算・解釈し、独立と無相関の違いを具体例で示す。

## I1-1 ★★（15分）
**学習目標**：ベイズと基準率。
保護リレー：$P(F)=0.01,\ P(\text{警報}\mid F)=0.99,\ P(\text{警報}\mid F^c)=0.05$。
1. $P(F\mid\text{警報})$ を求めよ。
2. 基準率を $P(F)=0.10$ に上げると $P(F\mid\text{警報})$ はいくつか。
3. （感度はそのままで）誤警報率を $0.05\to0.01$ に下げると $P(F\mid\text{警報})$ はいくつか。
4. 「基準率が低い領域では、感度を上げるより誤警報率（特異度）を下げる方が事後確率に効く」を、上の数値で確認・説明せよ。

## I1-2 ★★（10分）
**学習目標**：$P(A\mid B)\ne P(B\mid A)$。
I1-1 の数値で、$P(\text{警報}\mid F)$ と $P(F\mid\text{警報})$ を並べ、両者が大きく異なることと、その実務的含意（警報を見た運用員が陥りやすい誤り）を述べよ。

## I1-3 ★★（10分）
**学習目標**：独立 ≠ 無相関。
$X$ を $\{-1,0,1\}$ 上で各確率 $1/3$ にとり、$Y=X^2$ とする。
1. $E[X],\ E[XY]$ を計算し、$\mathrm{Cov}(X,Y)=0$（無相関）を示せ。
2. しかし $X$ と $Y$ は独立でないことを、$P(Y=1\mid X=0)$ と $P(Y=1)$ の比較で示せ。
3. この例の教訓を、電力（需要と再エネ出力を「無相関だから別々に」扱うことの危険）に結びつけて1–2文で。

## I1-4 ★★（10分）
**学習目標**：独立の計算判定。
サイコロ2個で $A=$「1個目が偶数」。$B_k=$「2個の和が $k$」（$k=7,8$）。
$A$ と $B_7$、$A$ と $B_8$ それぞれ独立か計算で判定し、「見た目の無関係さ」と「確率的独立」は別だと述べよ。
