# 演習（初級）★ — Module 0

> 学習目標：不確実性の表し方（観測値・予測値・分布・シナリオ）と、決定論が壊れる条件を**用語として**使えるようにする。

## B0-1 ★（5分）
**学習目標**：4つの解像度を区別する。
次の各記述は、観測値・予測値・確率分布・シナリオのどれか答えよ。
1. 「昨日の最大需要は 96 万kW だった」
2. 「明日の需要はだいたい 100 万kW でしょう」
3. 「明日の需要は 80 が確率0.5、120 が確率0.5」
4. 「需要が従う釣鐘型の曲線（平均100、ばらつき15）」

## B0-2 ★（5分）
**学習目標**：決定論が壊れる2条件。
「決定論モデルが壊れる2つの条件」を述べ、**両方が揃う**身近な例と、**揃わない（決定論で十分な）**例を1つずつ作れ。

## B0-3 ★（10分）
**学習目標**：偶然的／認識論的の分類。
次の不確実性を「偶然的（aleatory）」「認識論的（epistemic）」に分類し、理由を1文で。
1. 次の1秒間のPV出力の細かな揺らぎ
2. PVパネルの年あたり劣化率（正確な値を知らない）
3. サイコロの出目
4. ある地域の真の需要分布の平均（データが少なく不確か）

## B0-4 ★（5分）
**学習目標**：確率の役割の誤解を正す。
「確率を使えば未来が当てられる」は誤りである。確率は未来に対して**何を**与えるのか、1–2文で説明せよ。
