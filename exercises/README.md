# exercises/ — 演習問題

確率を「使える」状態にするための問題集です。**答えの数値より「なぜその確率表現を選ぶか」**を言語化することを重視します。

## 構成

```
exercises/
├── basic/         初級：定義の確認、1ステップ計算
├── intermediate/  中級：複数概念の接続、解釈
├── advanced/      発展：モデリング判断、電力応用、証明の直感
└── solutions/     解答・解説（全レベルを章別にまとめる）
```

ファイル名は章番号に対応：`00_why_probability.md`, `01_events_and_probability.md`, …

各 `basic/intermediate/advanced` ファイルは自己完結（問題＋メタ情報）。解答は `solutions/0X_*_solutions.md` に集約（standalone 版の追加問題は各解答末尾の「standalone 演習バンク」節）。

## 各問題のメタ情報

問題には次を明記します。

- **難易度**：★（初級）／★★（中級）／★★★（発展）
- **前提知識**：必要なノート・概念
- **学習目標**：その問題で何が身につくか
- **想定所要時間**

## 使い方

1. まず**自分の言葉で**答えを書く（数式の前に1文の説明）。
2. 計算は手で → 余裕があれば Python で検証。
3. `solutions/` で答え合わせ。**特に「採用した確率表現の理由」**を照合する。
4. 間違えた箇所は、対応ノートの「よくある誤解」表に戻る。

## 進捗の目安（合格ライン）

各章の「言えたら合格」（`roadmap/learning_map.md` §5）を、**人に説明できる**水準で。
