# 記法規約

この教材で使う記号を一覧にします。**すべてのノート・演習・ツールはここに従います。**
迷ったらこのページに戻ってください。記号の選び方そのものが、確率を理解する上での区別を反映しています。

---

## 1. 確率の基本

| 記号 | 読み・意味 | 注意 |
|---|---|---|
| $\Omega$ | 標本空間（起こりうる結果すべての集合） | 「世界の場合分けの全体」 |
| $\omega$ | 標本点（$\Omega$ の1要素、1つの結果） | $\omega \in \Omega$ |
| $A, B, \dots$ | 事象（$\Omega$ の部分集合） | $A \subseteq \Omega$。**集合**であることが要点 |
| $P(\cdot)$ | 確率測度 | 事象に $[0,1]$ の数を割り当てる関数 |
| $A^c$ | 余事象（補集合） | 「$A$ が起こらない」 |
| $A \cup B$ | 和事象 | 「$A$ または $B$」 |
| $A \cap B$ | 積事象 | 「$A$ かつ $B$」 |
| $P(A \mid B)$ | 条件付き確率 | 「$B$ が起きたと**分かった**もとでの $A$」 |
| $\varnothing$ | 空事象 | $P(\varnothing)=0$ |

## 2. 確率変数と分布

| 記号 | 読み・意味 | 注意 |
|---|---|---|
| $X, Y, Z$ | 確率変数（大文字） | $\Omega \to \mathbb{R}$ の**写像**。事象ではなく「結果に数を割り当てる関数」 |
| $x, y, z$ | 実現値（小文字） | $X$ が実際にとった具体的な数値 |
| $p_X(x)$ | 確率質量関数 PMF（離散） | $p_X(x) = P(X=x)$。**確率そのもの** |
| $f_X(x)$ | 確率密度関数 PDF（連続） | **確率ではない**（密度）。$f_X(x)\,dx$ がほぼ確率 |
| $F_X(x)$ | 累積分布関数 CDF | $F_X(x) = P(X \le x)$。離散・連続の両方で定義される |
| $X \sim \mathcal{D}$ | 「$X$ は分布 $\mathcal{D}$ に従う」 | 例：$X \sim \mathcal{N}(\mu, \sigma^2)$ |
| $\mathbb{1}[\cdot]$ | 指示関数 | 条件が真なら1、偽なら0 |

> **最重要の区別**：$p_X(x)$ は確率（足して1）、$f_X(x)$ は密度（**積分して**1）。
> 連続変数では $P(X=x)=0$、確率は区間の**面積** $P(a\le X\le b)=\int_a^b f_X(x)\,dx$ で決まる。

### よく使う分布の記号

| 分布 | 記号 | パラメータの意味 |
|---|---|---|
| 正規（ガウス） | $\mathcal{N}(\mu, \sigma^2)$ | $\mu$：平均、$\sigma^2$：分散 |
| 二項 | $\mathrm{Bin}(n, p)$ | $n$：試行回数、$p$：成功確率 |
| ポアソン | $\mathrm{Pois}(\lambda)$ | $\lambda$：単位時間あたり平均発生数 |
| 指数 | $\mathrm{Exp}(\lambda)$ | $\lambda$：率（平均は $1/\lambda$） |
| 一様 | $\mathcal{U}(a,b)$ | 区間 $[a,b]$ 上で一定 |

## 3. 期待値・ばらつき・関係

| 記号 | 読み・意味 | 一言 |
|---|---|---|
| $E[X]$ | 期待値（平均） | 「多数回繰り返したときの平均」 |
| $\mathrm{Var}(X)$ | 分散 | $E[(X-E[X])^2]$。散らばりの大きさ |
| $\sigma_X$ | 標準偏差 | $\sqrt{\mathrm{Var}(X)}$。$X$ と同じ単位 |
| $\mathrm{Cov}(X,Y)$ | 共分散 | 一緒に動くか |
| $\rho_{XY}$ | 相関係数 | $\mathrm{Cov}(X,Y)/(\sigma_X\sigma_Y)$、$[-1,1]$ |
| $E[X \mid Y]$ | 条件付き期待値 | 「$Y$ を知ったときの $X$ の平均」 |
| $q_\alpha(X),\ \mathrm{VaR}_\alpha$ | 分位点 / Value-at-Risk | 「下から $\alpha$ 番目の値」 |
| $\mathrm{CVaR}_\alpha(X)$ | 条件付きバリュー・アット・リスク | 「最悪 $(1-\alpha)$ の平均」＝尾部の平均 |

## 4. 不確実性下の最適化

ここで**記号の衝突**が起きます。意識的に使い分けてください。

| 記号 | 読み・意味 | 注意 |
|---|---|---|
| $x$ | **決定変数**（最適化で我々が選ぶ量） | 例：発電機出力、蓄電池充放電。確率の「実現値 $x$」とは別物 |
| $\xi$ | 不確実パラメータ（クサイ） | 例：需要、PV出力、価格。確率変数 or シナリオ or 集合の元 |
| $C(x,\xi)$ | コスト関数 | 決定 $x$ と不確実性 $\xi$ の両方に依存 |
| $g(x,\xi) \le 0$ | 制約 | 守るべき条件（潮流・需給バランスなど） |
| $\mathcal{U}$ | 不確実性集合（ロバスト最適化） | $\xi$ が動く「範囲」。分布ではなく集合 |
| $\mathbb{P}$ | 確率分布（$\xi$ の） | 真の分布。しばしば未知 |
| $\mathcal{P}$ | 分布の族＝曖昧性集合（分布ロバスト） | 「ありうる分布の集合」 |
| $\varepsilon$ | 許容違反確率（チャンス制約） | 例：$\varepsilon=0.05$ なら95%は制約を守る |
| $\alpha$ | 信頼水準（VaR/CVaR） | 例：$\alpha=0.95$ |

> ⚠️ **確率の $x$（実現値）と最適化の $x$（決定変数）は別の役割**です。
> この教材では、第0章–5 の確率パートでは $x$ を「実現値」として、
> 第6章の最適化パートでは $x$ を「決定変数」、不確実性は必ず $\xi$ として書きます。
> 混同しそうな箇所では、その都度ことわります。

## 5. 6つの最適化形式（早見表）

| 形式 | 定式化 | 不確実性の表現 | 何を最適化するか |
|---|---|---|---|
| 決定論的 | $\min_x f(x)\ \text{s.t.}\ g(x)\le 0$ | なし（点推定） | 1つのシナリオ |
| 期待値最小化 | $\min_x\ E[C(x,\xi)]$ | 分布 $\mathbb{P}$ | 平均 |
| 最悪ケース（ロバスト） | $\min_x\ \max_{\xi\in\mathcal{U}} C(x,\xi)$ | 集合 $\mathcal{U}$ | 最悪値 |
| チャンス制約 | $\min_x f(x)\ \text{s.t.}\ P(g(x,\xi)\le 0)\ge 1-\varepsilon$ | 分布 $\mathbb{P}$ | 制約違反確率 |
| CVaR最小化 | $\min_x\ \mathrm{CVaR}_\alpha(C(x,\xi))$ | 分布 $\mathbb{P}$ | 尾部リスク（最悪 $(1-\alpha)$ の平均） |
| 分布ロバスト | $\min_x\ \sup_{\mathbb{P}\in\mathcal{P}} E_{\mathbb{P}}[C(x,\xi)]$ | 分布の族 $\mathcal{P}$ | 最悪分布のもとでの平均 |

詳しい比較は [`optimization_map.md`](optimization_map.md) と `notes/06_optimization_under_uncertainty.md` を参照。

## 6. 表記上の約束

- ベクトルは太字にしない（文脈で判断）。必要なら成分を明示。
- 対数は自然対数 $\ln$。情報量で底2を使うときは明記。
- 「$:=$」は定義。「$\approx$」は近似。「$\sim$」は分布に従う。
- 図中の確率（面積）は塗りつぶし、密度（高さ）は曲線で区別する。
