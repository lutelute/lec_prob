# 学習地図

確率から確率的最適化までを、**概念の依存関係**として地図化します。
矢印 `A → B` は「A を理解していないと B でつまずく」を意味します。

---

## 2. 「不確実性をどう表すか」の5つの言語

この教材の背骨は、**同じ不確実性を5通りの言語で表せる**という点です。
章が進むにつれ、表現の解像度と用途が変わります。

```mermaid
graph LR
    R["現実の不確実性<br/>(需要・PV・価格・故障)"]
    E["① 事象<br/>A ⊆ Ω"]
    V["② 確率変数<br/>X: Ω→ℝ"]
    D["③ 確率分布<br/>PMF/PDF/CDF"]
    S["④ 有限シナリオ<br/>{ξ_s, π_s}"]
    U["⑤ 不確実性集合<br/>ξ ∈ 𝒰"]

    R --> E
    E --> V
    V --> D
    D --> S
    D --> U

    E -.->|第1章| L1[" "]
    V -.->|第2章| L1
    D -.->|第2-4章| L1
    S -.->|第5章| L1
    U -.->|第6章| L1

    style R fill:#fff4de,stroke:#8a5a10,color:#111
    style L1 fill:none,stroke:none
```

| 言語 | いつ使うか | 対応する章 | 最適化での役割 |
|---|---|---|---|
| ① 事象 | 「起こる／起こらない」の二分 | 第1章 | 故障の有無、ピーク発生 |
| ② 確率変数 | 結果に数値を割り当てたい | 第2章 | 需要量・出力量そのもの |
| ③ 確率分布 | ばらつき方の全体像が要る | 第2章–第4章 | 期待値・チャンス制約・CVaR |
| ④ 有限シナリオ | 分布を計算可能な形に落とす | 第5章 | シナリオベース確率計画 |
| ⑤ 不確実性集合 | 分布を仮定したくない／最悪に備える | 第6章 | ロバスト最適化 |

> **核心**：④は③を「捨てている」のではなく**離散近似**している。⑤は③の「形」を捨て、**範囲だけ**を残している。
> この違いが、期待値最小化・CVaR（③系）とロバスト最適化（⑤系）の分かれ目になります。

## 3. 「何を良しとするか」の4つの基準

最適化の形式は、結局**目的をどう測るか**で決まります。

```mermaid
graph TD
    Q["何を良い結果とみなすか？"]
    Q --> A["平均的に良い<br/>→ 期待値最小化"]
    Q --> B["悪いケースに強い<br/>→ ロバスト / 最悪値最小化"]
    Q --> C["制約をほぼ守る<br/>→ チャンス制約"]
    Q --> D["大損を避ける<br/>→ CVaR最小化"]

    A --> A2["E[C(x,ξ)]"]
    B --> B2["max_ξ C(x,ξ)"]
    C --> C2["P(g≤0) ≥ 1-ε"]
    D --> D2["CVaR_α(C)"]

    style Q fill:#fff4de,stroke:#8a5a10,color:#111
```

詳細は [`optimization_map.md`](optimization_map.md)。

## 4. 推奨学習ルート

```mermaid
graph LR
    subgraph 最短["最短ルート（骨格優先）"]
        a0[第0章] --> a1[第1章] --> a2[第2章] --> a5[第5章] --> a6[第6章]
    end
    subgraph 標準["標準ルート"]
        b0[第0章] --> b1[第1章] --> b2[第2章] --> b3[第3章] --> b4[第4章] --> b5[第5章] --> b6[第6章]
    end
```

- **最短**：まず「不確実性の表し方」と「最適化での使い方」を通す。期待値・分散の詳細は後追い。
- **標準**：積み上げ式。第3章（期待値・分散・相関）と第4章（データ）を飛ばさない。
- **応用逆算**：第6章冒頭の比較表を先に読み、未知語を第2章–第5章で回収。

## 5. 各章の「これが言えたら合格」チェック

| 章 | このひと言が自分の言葉で言えたら合格 |
|---|---|
| 第0章 | 「決定論モデルは ___ のとき壊れる。だから確率が要る」 |
| 第1章 | 「事象は $\Omega$ の部分集合、確率変数は $\Omega$ 上の関数。だから別物」 |
| 第2章 | 「$f_X(x)$ は確率ではなく密度。確率は区間の面積」 |
| 第3章 | 「平均が同じでも分散・CVaR が違えばリスクが違う」 |
| 第4章 | 「ヒストグラムは観測の要約、分布は母集団の仮定。両者は一致しない」 |
| 第5章 | 「シナリオは分布の離散近似。サンプルに極端事象が無いと尾部が見えない」 |
| 第6章 | 「この問題で私が ___ を重視するから、___ 形式を選ぶ」 |
